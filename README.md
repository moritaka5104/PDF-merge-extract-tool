PDF Merger, Sorter & Extractor

ブラウザ上で動作するセキュアなPDF編集ツールです。複数のPDFファイルの結合、特定ページの抽出・削除、ドラッグ＆ドロップによるページ並び替えを行い、新しいPDFファイルとして保存できます。

すべての処理はクライアントサイド（ブラウザ内）のJavaScriptで完結するため、PDFファイルが外部サーバーにアップロードされることはありません。

機能

PDF結合: 複数のPDFファイルをドラッグ＆ドロップで追加し、1つのファイルに結合できます。

ページ抽出・削除: 必要なページだけを選択（不要なページのチェックを外す）。

ページ並び替え: サムネイルをドラッグ＆ドロップして、ファイルやページの枠を超えて直感的に順序を変更。

ファイル名指定: ダウンロード時のファイル名を変更可能。

セキュアな処理: サーバー通信を行わず、プライバシーを保護。

使用技術

pdf-lib: PDFの生成・編集

PDF.js: PDFのレンダリング（サムネイル表示）

SortableJS: ドラッグ＆ドロップによる並び替え

使い方

index.html をWebブラウザで開きます（またはWebサーバー/GitHub Pagesにホストします）。

PDFファイルをドラッグ＆ドロップするか、クリックして選択します。

複数ファイル対応: 一度に複数のファイルを選択可能です。

追記: 既にリストが表示されている状態で、さらに別のPDFをドロップすると末尾に追加されます。

各ページのサムネイルが表示されます。

削除したいページ: チェックボックスを外します。

順序を変えたいページ: サムネイルをドラッグして移動させます。異なるPDFファイルのページ同士も混ぜることができます。

ファイル名を入力し、「作成してダウンロード」ボタンをクリックします。

Google Apps Script (GAS) での利用

Google Apps Scriptのウェブアプリとしてデプロイする場合：

GASプロジェクトを作成します。

index.html の内容をGASの index.html に貼り付けます。

コード.gs に以下の関数を追加します。

function doGet() {
  return HtmlService.createHtmlOutputFromFile('index')
    .setTitle('PDF結合・抽出ツール')
    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}


「デプロイ」>「新しいデプロイ」からウェブアプリとして公開します。
